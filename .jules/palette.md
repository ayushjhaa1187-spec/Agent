## 2024-05-24 - Accessibility Enhancements for `index.html`
**Learning:** Adding dynamic keyboard focus styling using `currentColor` and `outline-offset` provides excellent visual feedback across varying background colors. Furthermore, decorative elements with emoji icons (like `.feature-icon`) must be explicitly hidden from screen readers using `aria-hidden="true"` to prevent repetitive or confusing announcements when surrounding text already provides full context. Smooth scrolling also needs to be wrapped in `@media (prefers-reduced-motion: no-preference)` to respect user OS accessibility settings.
**Action:** When creating focus outlines in this project, use `currentColor` to dynamically contrast against varying background gradients. Always apply `aria-hidden="true"` to structural emojis or decorative icon spans to streamline the accessibility tree.
