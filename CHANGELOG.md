# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-04-06

### Added
- Light and dark mode with system preference detection and manual toggle
- Custom 404 page
- Sticky navbar with dark mode toggle and branding
- Footer with copyright and links
- Skeleton loaders for data fetching states
- Empty states with contextual messages
- `.env.example` with all required environment variables
- `.prettierrc` configuration
- `CHANGELOG.md`

### Changed
- Replaced neon cyber theme with professional warm neutral design system
- Updated color palette: light (#f7f6f2 bg, #01696f primary) and dark (#0f0f0f bg, #4f98a3 primary)
- Loaded Inter font via `next/font` for consistent typography
- Improved README with accurate project structure, tech stack, and setup guide
- Cleaned up console.log statements

### Fixed
- Preview panel now respects dark mode
- Welcome modal styling updated for both themes

## [0.1.0] - Initial Release

### Added
- AI-powered code generation with multi-model support
- Vercel Sandbox integration for live execution
- Split-panel IDE layout (chat, preview, file explorer, logs)
- Auto error detection and fix reporting
- Mobile responsive tab layout
