## ✨ Features
| Feature | Description |
| :--- | :--- |
| **AI Pair Programming** | Real-time AI code suggestions powered by OpenAI, Claude, and Gemini |
| **Vercel Sandbox** | Execute code in isolated cloud sandboxes with instant preview |
| **Multi-Model Router** | Dynamically switch between GPT-4o, Claude Opus, and Gemini Pro |
| **Resizable Panels** | Split editor, preview, and chat panels in a customizable layout |
| **Syntax Highlighting** | Full language support with React Syntax Highlighter |
| **State Management** | Zustand-powered reactive state for seamless UX |

## 🛠 Tech Stack
| Layer | Technology |
| :--- | :--- |
| Framework | Next.js 16 (Turbopack) |
| Language | TypeScript |
| AI | Vercel AI SDK (OpenAI, Anthropic) |
| UI | Radix UI, Tailwind CSS v4, Lucide |
| State | Zustand |
| Runtime | Vercel Sandbox |

## 🚀 Quick Start
```bash
git clone https://github.com/ayushjhaa1187-spec/vibe-coding-platform.git
cd vibe-coding-platform
npm install
npm run dev
```
